  • Question 01: Differences among the map, filter, find, forEach.

  • Ans: => map() perform the same code on every element in an array. It return a new array with updated elements.
    => filter() checks every element in an array to see if it meets a certain criteria. It returns a new array with the elements that return truthy for the criteria.
    => forEach() is used to execute the same code on every element in an array but doesn't change the array and it returns undefined.
    => find() returns only one match in an array.

  • Question 02: Differences among the localStorage, sessionStorage and Cookies?

  • Ans: => local storage- it is beneficial for storing data that the user will need to access later, such as: offline data.
    => Session storage- it is a very useful way to improve the performance of one's web applications.
    => Cookies- Cookies are a good choice for storing data that shouldn't be persisted for a long time.
